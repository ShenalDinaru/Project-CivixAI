🚀 QUICK START (5 STEPS)
════════════════════════════════════════════════════════════════════════════

Step 1: Create Firebase Project
   → Go to https://firebase.google.com
   → Create new project
   → Enable Realtime Database
   → Download service account key

Step 2: Install Backend Dependencies
   → Open terminal/command prompt
   → cd backend
   → npm install

Step 3: Create .env File
   → In backend folder, create .env
   → Copy values from service account JSON:
      PORT=5000
      FIREBASE_PROJECT_ID=your-project-id
      FIREBASE_PRIVATE_KEY=your-private-key
      FIREBASE_CLIENT_EMAIL=your-email
      FIREBASE_DATABASE_URL=https://your-project.firebaseio.com

Step 4: Configure Firebase Rules
   → Firebase Console → Realtime Database → Rules
   → Copy and paste the provided rules

Step 5: Start Backend
   → npm start
   → Server runs on http://localhost:5000 ✓

🔌 API ENDPOINTS
════════════════════════════════════════════════════════════════════════════

GET /api/health
   → Check if server is running
   → Response: {"message":"CivixAI Backend is running"}

POST /api/auth/signup
   → Register new user with complete data
   → Requires: firstName, surname, username, email, phone, password
   → Returns: success, user data, or error message

POST /api/auth/check-username
   → Check if username is available
   → Requires: username
   → Returns: available (true/false)

POST /api/auth/check-email
   → Check if email is available
   → Requires: email
   → Returns: available (true/false)
